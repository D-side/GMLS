The idea of this site was initially to keep a list of weird GML snippets, similar to a way WTFJS does this with JavaScript. But since GML is a much simpler language, there aren't that many WTFs to be discovered.

So this site is listing any useful snippets

WTF above is an abbrevation of "What a Terrible Failure", and means a situation where the language acts in a weird manner.
